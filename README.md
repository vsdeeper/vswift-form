# vswift-form

A simple but not simple form designer based on [Vue](https://cn.vuejs.org/) and [Element Plus](https://element-plus.org/).

## Develop

```sh
pnpm dev
```

## Build lib mode

```sh
pnpm build:lib
```

It will emit `dts` for components, then build output `.js` and `.cjs` file.

## Publish

```sh
pnpm release
```
